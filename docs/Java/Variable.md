# Variable

Reference
### 변수 초기화
멤버변수(인스턴스변수)는 초기화를 하지 않아도 자동적으로 자료형에 맞는 기본값으로 초기화가 이루어짐. 지역변수는 사용하기 전에 반드시 초기화를 해야한다.

클래스 초기화 블럭, 인스턴스 초기화 블럭: 각 클래스변수/인스턴스 변수의 복잡한 초기화에 사용.
생성자보다 인스턴스 초기화 블럭이 먼저 수행된다.
